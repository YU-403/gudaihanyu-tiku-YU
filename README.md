# 《古代汉语》AI题库————基于王力《古代汉语》教材编写的AI生成题库HTML网页端
✨ 功能特色
答题操作
步骤	操作
①	点选你认为正确的选项（单选题点一个、不定项可选多个、判断题点「正确」或「错误」）
②	点击「确认答案」提交
③	查看正确答案标示与解析说明
④	点击「下一题」继续
三种题型
单选题 — 四选一，点选后确认
不定项选择题 — 可选多个选项，全部选对才计正确
判断题 — 点选「正确」或「错误」后确认
交互特色
音效反馈 — 点选选项有短促点击音，答对有上升双音，答错有低沉蜂鸣
视觉动效 — 选项悬停放大、确认按钮按压反馈、结果面板逐项揭开、页面切换滑入转场
连续答对提示 — 连续答对 3 题后显示「🔥 N 连对」
进度条平滑动画 — 专题/生涯模式进度条平滑增长
完成页数字滚动 — 完成统计时数字从 0 递增滚动到最终值
数据与隐私
所有数据存储在浏览器本地（localStorage），不经过任何服务器
无需注册，无需登录
无后端，纯前端 HTML 单页应用
关闭页面后重新打开，进度仍然保留
🔄 如何更新
本题库会持续扩充和修正。每次更新后，打开页面刷新即可获得最新版本。

本地更新（自建）
如果你需要在本地修改或扩充题库：

修改 常用词部分/、通论部分/、文选部分/ 目录下的 .md 文件
双击运行 build.bat
重新生成的 index.html 即包含最新题库内容
🛠 技术栈
前端：纯 HTML / CSS / JavaScript（无任何框架或库）
音效：Web Audio API（无外部音频文件）
数据持久化：浏览器 localStorage
构建脚本：Python 3（标准库，无第三方依赖）
公式渲染：（本项目不使用，但模板支持 KaTeX）
部署：GitHub Pages
📁 项目结构
古汉语AI题库/
├── index.html          # 刷题网站（单页自包含，双击可用）
├── build.py            # 构建脚本（解析 .md 生成 index.html）
├── build.bat           # Windows 一键构建
├── README.md           # 本文件
├── 常用词部分/         # 常用词题库（.md 格式）
├── 通论部分/           # 通论题库（.md 格式）
└── 文选部分/           # 文选题库（.md 格式）
