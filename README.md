# 《古代汉语》文选、常用词、通论 AI 刷题库

> 王力《古代汉语》配套题库 · 纯前端 · 免后端 · 免注册

本项目是一个基于王力《古代汉语》（校订重排本）教材的 AI 辅助刷题网站，涵盖文选、常用词、通论三大板块，共 **1141 道题**（持续扩充中）。所有数据保存在浏览器本地，无需后端服务器，无需注册账号。

---

## 📖 简介

本题库基本按照课本的已学部分编制。工作流以网上搜集的题目和自己编写的题目作为样本提示，编写提示词，让大语言模型接入含有整理过课本原文的 RAG（检索增强生成）系统，仿照样本形式生成题目、正确答案、详细解析与知识回溯条目。

由于使用大模型生成，外加处理原始 PDF 教材时使用了 OCR（MinerU），题目表述或答案解析中可能出现少量错误，敬请理解。如发现错误或有建议，欢迎反馈。

## 🎮 刷题模式

### 📖 专题模式

按三大板块（常用词／通论／文选）及其下子章节选择题目范围，进入后依次顺序答题。答完该章节全部题目后显示正确率统计。

- 中途按「← 返回」退出时会**自动保存当前进度**（答题位置、正确与错误题数）
- 下次进入同一子章节时会弹窗询问是否继续上次进度

### 🎲 随机无尽模式

从所有题库中随机抽取题目，答完一题自动切换至下一题，永不重复。退出后进度不保留，适合碎片时间快速自测。

### 🏆 生涯模式

所有题目随机排序依次出现。答对的题目**永久移除**，答错的题目移入错题池，待当前轮次完成后错题池重新打乱形成新一轮，直到全部题目都被答对才算通关。

- 每次答完一题即**自动存档**（localStorage）
- 关闭页面或中途退出后重新进入，直接从退出时的题目继续

## ✨ 功能特色

### 答题操作

| 步骤 | 操作 |
|:----:|:------|
| ① | 点选你认为正确的选项（单选题点一个、不定项可选多个、判断题点「正确」或「错误」） |
| ② | 点击「确认答案」提交 |
| ③ | 查看正确答案标示与解析说明 |
| ④ | 点击「下一题」继续 |

### 三种题型

- **单选题** — 四选一，点选后确认
- **不定项选择题** — 可选多个选项，全部选对才计正确
- **判断题** — 点选「正确」或「错误」后确认

### 交互特色

- **音效反馈** — 点选选项有短促点击音，答对有上升双音，答错有低沉蜂鸣
- **视觉动效** — 选项悬停放大、确认按钮按压反馈、结果面板逐项揭开、页面切换滑入转场
- **连续答对提示** — 连续答对 3 题后显示「🔥 N 连对」
- **进度条平滑动画** — 专题/生涯模式进度条平滑增长
- **完成页数字滚动** — 完成统计时数字从 0 递增滚动到最终值

### 数据与隐私

- **所有数据存储在浏览器本地**（localStorage），不经过任何服务器
- **无需注册**，无需登录
- **无后端**，纯前端 HTML 单页应用
- 关闭页面后重新打开，进度仍然保留
